# 🎓 LMS Portal

A modern Learning Management System (LMS) built with React.js, Vite, Firebase Authentication, and Tailwind CSS.

## 🚀 Features

* User Registration & Login
* Firebase Authentication
* Explore Courses
* Course Details Page
* Course Enrollment & Unenrollment
* Student Dashboard
* Progress Tracking
* Certificate Generation
* Admin Panel
* Add & Delete Courses
* Responsive Design

## 🛠️ Technologies Used

* React.js
* Vite
* Tailwind CSS
* Firebase Authentication
* React Router DOM
* Local Storage

## ⚡ Installation

```bash
git clone https://github.com/yourusername/lms-portal.git

cd lms-portal

npm install

npm run dev
```
